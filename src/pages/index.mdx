# WalletD API Documentation

WalletD is an crypto library designed to make using and developing for different cryptocurrencies easy. 
WalletD supports the handling of mnemonic phrases, hierarchical deterministic (HD) crypto wallets, connecting to decentralized blockchains, and instantiating/importing multi-crypto wallets to check balances and transact.
