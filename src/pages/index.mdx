# WalletD API Documentation

⚠️ This project is under heavy development. Expect bugs & breaking changes prior to v1.0.0's release.

## What is WalletD?

WalletD is an ecosystem designed to make using different cryptocurrencies easy. Built in Rust, we plan to make our [official crates](https://crates.io/crates/walletd) available for use on multiple platforms through the use of WASM libraries.

## Status
<Note>
  ⚠️ This project is under heavy development. Expect bugs & breaking changes prior to v1.0.0's release.
</Note>

Supports (or should support)

- Generation of mnemonic phrases
- Importing of mnemonic phrases
- Generation of Hierarchical Deterministic (HD) wallets
- Importing of HD wallets
- Support for Bitcoin, Ethereum, and other blockchains
