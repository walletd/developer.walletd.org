# What is WalletD?

WalletD is an SDK designed to make using different cryptocurrencies easy. We plan to make our [official Rust crates](https://crates.io/crates/walletd) available for use on multiple platforms through the use of WASM libraries.

Supports (or should support)
- Generation of mnemonic phrases
- Importing of mnemonic phrases
- Generation of Hierarchical Deterministic (HD) wallets
- Importing of HD wallets
- Support for Bitcoin, Ethereum, and other blockchains
