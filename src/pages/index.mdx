# WalletD Developer Documentation

⚠️ This project is under heavy development. Expect bugs & breaking changes prior to v1.0.0's release.
## What is WalletD?
WalletD is an ecosystem designed to make using different cryptocurrencies easy. Built in Rust, our official crates are made available for use on multiple platforms through the use of WASM libraries.
